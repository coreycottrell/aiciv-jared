# Logo No-Overlap Rule — CONSTITUTIONAL (Jared 2026-04-15)

**Type**: teaching (transferable wisdom for every future image)
**Topic**: Logo placement — hexagon + PUREBRAIN.ai wordmark NEVER overlaps main image content

## The rule

Logo (hexagon + PUREBRAIN.ai wordmark) must NEVER overlap:
- Faces / headshots / people
- Product shots / focal subjects
- Primary text / headline blocks
- The eyeline-to-headline visual path

**Minimum 80px logo-to-subject distance.**

**Dense image → logo gets its OWN dedicated band** (top 10% or bottom 10% strip), not floating over the work.

## Why

Overlap = visual noise that chases attention at the cost of resonance. Per PT principle "engineer resonance, don't chase attention." Overlap damages brand, competes with message, reads as lazy layering.
