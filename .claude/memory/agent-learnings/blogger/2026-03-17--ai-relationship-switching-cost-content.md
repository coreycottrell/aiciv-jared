## The Core Argument

When you switch AI tools, you lose the accumulated relationship — the trained preferences, learned vocabulary, and pattern the system built of how you work. Unlike documents or templates, this contextual layer is not portable. No standard format. No import/export. What develops inside a platform stays inside that platform.

This is the new vendor lock-in: not infrastructure lock-in, relational lock-in.

Anchor stats used:
- 15–30 min productivity loss per person per context switch (Plurality Network 2026)
- 42% higher satisfaction + 31% better output quality for workers whose AI "knows their context" (Salesforce State of Work 2025)
- Gartner 2026: "AI relationship depth" named as emerging competitive differentiator
- McKinsey: model first-mover advantage = 6–8 weeks (from prior post research, consistent across sources)
