## What the Skill Does

Universal reasoning unblock tool. When any agent gets stuck in a loop:
- STOP generating options
- NARRATE the problem out loud in plain language (as if explaining to a non-technical human)
- LISTEN for contradictions, surfaced assumptions, obvious choices
- RESUME with clarity

Key insight: "The explanation and the breakthrough are the same moment."

## When to Use It

- Circular reasoning (same options 3+ times)
- Stuck 2+ turns without progress
- Can't choose between two valid paths
- About to ask Jared a question (narrate first)
