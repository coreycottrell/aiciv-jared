## Key Patterns Discovered

### 1. Direct Comparison Anchoring

When differentiating from known products (ChatGPT, Claude), direct comparison in the opening line works better than abstract benefit statements.

**Pattern**: "Unlike [known competitor], your [product]:"

This anchors the reader in familiar territory, then pivots to differentiation. Readers don't need to figure out what category you're in.

### 2. The Four Differentiators Stack

For PureBrain, these four points form a coherent value proposition:
1. Memory persistence (solves repetition pain)
2. Autonomous work (solves "always on call" pain)
3. Personalized learning (solves "generic responses" pain)
4. Named relationship (creates psychological commitment)

The fourth one (naming) is the most emotionally powerful because it transforms a tool into a partner.

### 3. Video Script Structure: Problem-Solution-Proof

Each video followed:
- **Hook** (0-10 sec): Familiar pain point
- **Problem visualization** (10-20 sec): Show the frustration
- **Solution** (20-40 sec): Show the relief
- **Close** (40-60 sec): Memorable tagline

This structure works because it starts with what the viewer already feels, not what the product does.

### 4. Personalization Placeholder "[AI Name]"

Using "[AI Name]" in scripts serves two purposes:
1. Technical: Placeholder for dynamic personalization if supported
2. Psychological: Reminds viewer that their AI has a name (reinforces differentiation)

### 5. Tone Calibration for Pure Technology

Per brand voice:
- Confident, not arrogant
- Factual, not salesy
- "Your AI remembers" (factual) vs "World's best AI" (arrogant)

## Reusable Templates

### Differentiation Bullet Format
```
Unlike [Competitor A] or [Competitor B], your [Product]:

- [Benefit that solves pain 1]
- [Benefit that solves pain 2]
- [Benefit that solves pain 3]
- [Emotional differentiator]
```

### Video Script Skeleton
```
OPENING (0-10 sec)
[Visual: Familiar pain point]
VOICEOVER: "You've probably experienced..."

PROBLEM (10-20 sec)
[Visual: Pain visualization]
VOICEOVER: "And this. And this."

SOLUTION (20-40 sec)
[Visual: Product solving the pain]
VOICEOVER: "Your [Product] does this differently..."

CLOSE (40-60 sec)
[Visual: Clean tagline frame]
VOICEOVER: "[Memorable summary]"
[Text: "Key message."]
```
