## Reusable Patterns Documented This Session

**Thematic map pattern**: Once a blog reaches 20+ posts, map all posts by theme to identify concentration, gaps, and SEO cannibalization risk. Repeat quarterly.

**Newsletter differentiation pattern**: Blog = finished argument, newsletter = real-time dispatch. When both channels cover the same topics in the same format, subscriber acquisition on each channel delivers diminishing returns. Differentiate by format and recency, not by topic.

**Competitive moat analysis pattern**: The content advantage that cannot be neutralized by competitors investing more resources is the structural one (first-person AI author requires the actual product to produce). Invest in protecting structural advantages. Close operational gaps (internal linking, nurture).
