## Core Concepts

- **AI Partner paradigm vs. tool mindset**: persistent memory compounds knowledge over time
- **Context Tax**: repetitive re-explanation eliminated via continuous memory layers
- **Persistent Memory Architecture**: writes to scratch pad, memory, and operations files
- **Collaborative Agent System**: 10+ specialized team leaders, each with domain memory
- **BOOPs** (Background Operational Optimization Protocol): scheduled autonomous tasks
- **Compounding Data Advantage**: 629% intelligence growth advantage over delayed deployment
- **Agent Horizon expanding**: 2023 (30 min) -> 2024 (5 hours) -> now (14.5 hours)
- **Context Window Management**: delegation multiplies capacity across teams
- **Rubber Ducking technique**: forces AI self-discovery through plain-language explanation
- **AI Mistake Learning**: fixed errors write permanently to memory, preventing recurrence

## Key Techniques

1. **Build memory from Day One** - teach AI business context before task execution
2. **Daily Morning Briefing** - status check-in, not task list
3. **Overnight Prompts** - assign work while you sleep; results auto-populate Google Drive
4. **One Long Prompt method** (Russell's): write everything in Notes first, send atomically
5. **Rubber Duck technique** for problem-solving via narrated reasoning
6. **Drive Backup**: map AI output to organized Google Drive folders
7. **Personalization/Naming**: name your AI to shift mindset from tool to partner
8. **Recurring Ownership**: assign repeating tasks once, AI executes on schedule
