## Core Concepts

- **"Prompting is Dead"**: "Prompts give you speed, workflows give you leverage"
- **Workflow vs. Prompt distinction**: repeatable systems vs. one-time asks
- **Workflow Candidate Signals**: repeatable, predictable, rule-based, expert-level tasks
- **Three Levels of AI Workflow**: Manual Chain (L1) -> Semi-Automated (L2) -> Fully Automated (L3)
- **Context Window Multiplication**: delegation creates new 170K-token windows (total 1.2M+)
- **BOOPs as Workflow Infrastructure**: schedule autonomous delegation and optimization
- **Five-Step Process Mapping**: Name -> Inputs -> Steps -> Decisions -> Output Definition
- **What Kills Workflows**: over-automating too fast, no checkpoints, skipped mapping
- **Voice AI Integration**: 11 Labs API enables zero-code voice automation
- **Real Business Examples**: onboarding, content creation, lead qualification, SEO optimization

## Key Techniques

1. **Five-Step Process Map**: pick one recurring task, document every micro-step
2. **Level Ladder**: start Level 1 (manual), graduate to Level 2, then Level 3 only for low-stakes
3. **Russell's Cardinal Rules**: (1) CC me on outgoing email always, (2) Acknowledge + plan first, (3) Show your thinking
4. **Delegation as Context Amplifier**: large tasks -> new context window -> 170K becomes 1.2M+
5. **Ask AI to Build Its Own BOOPs**: "How would you optimize yourself with scheduled tasks?"
6. **Data Ingestion approach**: ingest 19 years of emails, courses, YouTube, meetings, SOPs
7. **One Workflow Challenge**: identify and map the first automation target right now
8. **Process-First Rule**: "You can't automate what you haven't fully defined"
