# Boardy.ai Competitor Research — Key Learnings

**Date**: 2026-03-09
**Type**: competitive-intelligence
**Topic**: Boardy AI superconnector — startup ecosystem networking product

## Core Finding

Boardy is NOT a PureBrain competitor. It is a startup-ecosystem-specific introduction platform powered by AI phone calls. Single use case (warm intros), episodic (not persistent), free (data-monetized).

## Business Model

Free to users. Monetizes via: (1) hashed contact data shared with ad platforms, (2) community partnership fees (27+ communities), (3) Boardy Ventures co-investment carry. Terms of service explicitly states data sharing with advertising platforms.

## The Community Partnership Playbook

Boardy has 27+ dedicated landing pages for specific communities: YC, Buildspace, McKinsey alumni, OpenAI community, Next36, Slush, etc. Each community brings hundreds of pre-qualified professionals into the Boardy matching pool. This is a B2B2C distribution model — extremely efficient. REPLICABLE for PureBrain via mastermind groups, coaching programs, professional associations.

## Voice AI is Production-Ready

Boardy's core mechanism is a phone call from an AI. This is live, operational, and users accept it. Voice-based AI for professional contexts is not future tech — it's here. PureBrain should prototype a voice check-in feature.

## Episodic vs Continuous AI

Boardy is episodic — each call stands alone, AI doesn't remember you between uses. PureBrain is continuous — memory compounds. This is a genuine, defensible differentiator. Use this in PureBrain marketing copy.

## Data Trust Gap

Boardy's "free" model means user data funds the product. PureBrain's subscription model means users own their data. This is a material trust advantage that should be communicated explicitly.
