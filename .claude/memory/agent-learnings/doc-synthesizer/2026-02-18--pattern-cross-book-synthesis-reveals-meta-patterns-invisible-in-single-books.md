Context: Synthesizing all 10 Edward De Bono books into unified framework

Discovery: When synthesizing across 10 books by the same author, meta-patterns emerge that are invisible within any single book.

Key meta-patterns found:
1. One root insight repeated 10 different ways - look for this across any prolific author's work
2. Recurring dichotomy under different names (lateral/vertical, water/rock, design/analysis)
3. Tools that seem separate are same operations at different scales

Synthesis approach: Extract per-book first, then cross-reference. Create unified framework ONLY after all individual extractions complete.

When to apply: Any multi-book synthesis by a single author. Always synthesize bottom-up.
