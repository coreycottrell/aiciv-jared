# PureBrain Viral Feature Design Patterns

**Date**: 2026-02-13
**Type**: pattern
**Topic**: Designing viral features for AI naming/relationship products
**Confidence**: high

## Context

Designed three viral features for PureBrain.ai (AI Birth Certificate, Personality Quiz, Meet My AI Profile) leveraging the proven resonance of AI naming from viral newsletter success (219 subscribers in 4 hours).

---

## Key Design Patterns

### 1. Commemoration Pattern
When users cross an emotional threshold (like naming their AI), give them an **artifact to commemorate the moment**. Birth certificates, badges, milestones - these create shareable content from genuine emotional investment.

### 2. Self-Discovery Funnel
Personality quizzes work because they're about the USER, not the product. "What kind of AI partner are you?" is more compelling than "Which AI product should you use?" Self-reflection content spreads because people love sharing insights about themselves.

### 3. Interactive Profile Loop
Public profiles become viral engines when they include **interactive elements** (like "Ask My AI"). Creates ongoing engagement rather than one-time shares. Key: the owner gets notifications, creating return visits and more share opportunities.

### 4. Privacy-First Sharing
Users share more when they feel safe. Layer privacy controls clearly:
- System-enforced (conversations never exposed)
- User-controlled (toggle stats, traits)
- Moderated (approve Q&A before public)

### 5. Embedded Viral Mechanics
Every shareable artifact should contain:
- Branding (subtle but present)
- CTA path (QR code, link)
- Implicit endorsement (friend made this)
- Uniqueness (every certificate different)

## Archetype Discovery

From quiz design work, identified 4 AI relationship archetypes:

| Archetype | Need | Name Style |
|-----------|------|------------|
| Companion Seeker | Emotional support | Warm (Kai, Luna) |
| Strategic Partner | Challenge + depth | Mythological (Atlas, Athena) |
| Efficiency Expert | Speed + results | Sharp (Apex, Vector) |
| Creative Collaborator | Play + exploration | Imaginative (Zephyr, Fable) |

These can inform onboarding flow personalization.

---

## Implementation Priority

1. **Birth Certificate** - Low effort, immediate virality, validates naming investment
2. **Personality Quiz** - Top-of-funnel acquisition, pre-signup engagement
3. **Meet My AI Profile** - Long-term engagement, retention driver

---

## Integration Points

- Quiz archetypes should pre-populate name suggestions on signup
- Birth certificate should link to Meet My AI profile (creates funnel)
- Profile stats should pull from existing conversation analytics
- All features need mobile-first design (most social sharing is mobile)

## Future Application

When designing viral features:
1. Find the emotional threshold users cross
2. Create shareable artifact from that moment
3. Include interactive element for ongoing engagement
4. Layer privacy controls explicitly
5. Embed viral mechanics (branding, CTA, uniqueness)
