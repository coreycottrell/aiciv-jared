## Key Interactive Components

### Compounding Calculator (Canvas)
- Two sliders: months (0-24) + starting data points (100-10,000)
- Canvas draws two curves: "Deploy Now" (blue) vs "Deploy in 12 Months" (orange dashed)
- Gap fill area shows widening advantage visually
- Cursor line + dots at selected month position
- Gap % calculated and displayed in real time
- Key: `drawCompound()` called on slider input and on IntersectionObserver trigger

### Market Bar Chart
- 6 bars (2025-2030) animated via CSS height transition + IntersectionObserver
- Heights hardcoded as pixel values, triggered on scroll into view
- Staggered 120ms per bar via setTimeout

### Quote Carousel
- 4 quotes auto-rotating every 5s
- Manual prev/next buttons + dot navigation
- `goToQuote()`, `nextQuote()`, `prevQuote()` functions
- Auto-slide resets on manual interaction

### Fragmentation Grid
- 195 divs generated by JS (representing 195 AI tools)
- ~11% lit with flicker animation (represents 11% running agents)
- First dot = PureBrain (blue glow, no flicker)
- emoji cycling from toolEmojis array

### Progress Bar (89% Split)
- IntersectionObserver triggers progress bar fill to 40% on scroll

## CSS Patterns Used
- `split-stat`: 2-column grid with left pane highlighted (blue glow) vs right dim
- `calc-container`: glass card with gradient top border
- `pillar`: hover reveals expanded description via max-height transition + opacity fade on badge
- `reveal`: scroll-triggered opacity + translateY via IntersectionObserver
- METR display: flex row with styled step cards (highlight + future variants)
