## What Cursor Is

AI-powered code editor. Sits inside your IDE, understands your codebase semantically, provides intelligent autocomplete (Tab model), agentic coding workflows (Agent mode), and multi-model support (GPT-5, Claude, Gemini, Grok, Cursor proprietary). Integrates with GitHub (BugBot PR reviews) and Slack. Used by over half of Fortune 500. SOC 2 certified. Free Hobby tier; Pro at $20/mo; Pro+ $60/mo; Ultra $200/mo; Teams $40/user/mo.

**Key insight**: Cursor is a DEVELOPER TOOL, not a business AI partner. It accelerates code writing — not business strategy, marketing, ops, or institutional knowledge.

---

## Comparison Framing Strategy

The honest framing used in this page: "These are different products for different jobs." Cursor wins on: code autocomplete, codebase understanding, agentic coding, multi-model flexibility, GitHub PR review, free entry tier. PureBrain wins on: persistent memory, full business scope, non-developer access, institutional knowledge, compounding intelligence, named AI identity.

The key competitive insight is SCOPE — Cursor covers code, PureBrain covers the entire business.
