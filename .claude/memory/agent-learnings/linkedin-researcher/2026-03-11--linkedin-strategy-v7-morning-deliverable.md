### Algorithm: First Comment Link Penalty (NEW - Critical)

Dataslayer February 2026 report confirmed: the workaround of putting links in the first comment is now penalized by the LinkedIn algorithm. This changes prior recommendations in v1-v6. New rule: no links anywhere in the first 90 minutes. Links after the Golden Window are lower-risk but still risky.

Buffer 2026 data shows late afternoon/evening (3-8 PM) now competes with morning slots. Prior data was 2025-era. Wednesday 4 PM ET remains best. Evening posting is now equally valid.
