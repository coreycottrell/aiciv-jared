## Campaign Architecture

- Days 2-5: Stat cards 1-4 drip (organic only)

---

## Key Algorithm Patterns Applied (2026 Knowledge Base)

1. Post once per day max — back-to-back posts cannibalizes by 20%
2. Dwell time is now a ranking factor — long-form captions (1,300-1,800 chars) win
3. Comments from authority figures (CEOs, VCs) have the most algorithmic impact
4. 3+ links in a post performs 210% better (links are no longer penalized when value-add)
5. Replies in first hour extend dwell time — reply to every comment with a NEW insight
6. Post organically first, then start boost 15-20 min later (seeds organic signal before paid)
7. LinkedIn newsletters run on a SEPARATE algorithm from the feed — sending on same day as feed post does NOT cannibalize
