## Key Insights

### 1. New Domain Baseline vs Problem Signal

These numbers are NOT failure indicators — they are the expected baseline for a domain launched in February 2026 with no prior history. The correct mental model: these numbers only go up from here.

### 2. Highest-ROI Actions in Priority Order

1. GSC + Bing sitemap submission (site not indexed = zero organic traffic possible)
2. Meta descriptions + OG tags (missing on all posts = lost social traffic)
3. H1 fix on homepage (Elementor bypasses semantic heading hierarchy)
4. Internal cross-links between all blog posts (85% internal linking score = easy win)
5. Noindex dev/test pages (pay-test pages waste crawl budget + hurt credibility)
6. Directory submissions (fast backlink foundation, mostly free)
7. Core Web Vitals fixes (WebGL + Elementor = mobile performance bottleneck)
8. GEO/FAQ schema (AI visibility requires separate optimization from traditional SEO)

### 3. GEO Is a Separate Discipline from SEO

The overlap between Google's top 10 results and AI citations (ChatGPT/Perplexity) is now below 20%. Must optimize for both separately. Key GEO requirements:
- Answer-first content structure
- FAQ schema on every post (Perplexity actively scrapes this)
- Definitional pages ("What Is AI Partnership?") that AI systems extract
- Allow AI crawlers in robots.txt (GPTBot, PerplexityBot, Claude-Web)
- Multi-source presence (directories, social, guest posts all reinforce AI citations)

### 4. Existing Media Coverage = Fastest Backlinks

Jared has unlinked mentions in Thrive Global, Jerusalem Post, CEOWORLD. Email each requesting URL update to purebrain.ai — takes 30 minutes, delivers DA 70-90 editorial backlinks with no content creation required.

### 5. Directory Submission Priority Order

For a consulting/partnership model (not pure SaaS):
1. Clutch.co (DA 89) — requires client reviews to rank, submit immediately
2. GoodFirms (free) — AI consulting category, gets cited by Perplexity
3. aitools.fyi ($20 fast-track) — 40% CTR, best ROI
4. There's An AI For That (free or $347) — DR 76
5. Product Hunt — plan a full coordinated launch, not a passive submission

### 6. Core Web Vitals Bottlenecks for This Stack

Confirmed stack: WordPress + Elementor + WebGL/Three.js. Main bottlenecks:
- Three.js on mobile → mobile score 35-55 range without fix
- Elementor default loads 15-20 JS files per page → enable Improved Asset Loading
- No caching layer → install WP Rocket ($59/year)
- Images not in WebP format → ShortPixel plugin (100 images free/month)

### 7. Keyword Strategy Foundation

Target 20 keywords mapped to 5 intent buckets:
- Awareness: "why AI projects fail", "enterprise AI adoption failure"
- Consideration: "AI partner vs AI tool", "persistent AI memory", "AI that remembers context"
- Decision: "AI business partner", "ChatGPT alternative with memory"
- Long-tail: "why do AI chatbots forget everything", "AI tools for consultants that remember clients"
- Brand: "PureBrain AI", "purebrain.ai" (monitor, not target)

Long-tail keywords are where a new domain can rank in weeks, not months.
