## Context

Designed complete pricing architecture for PureBrain's "Have Us Fix Everything" execution service — the upsell attached to the $47 AI Website Analysis report. This is a new revenue layer built on near-zero marginal cost delivery (AI agents execute all fixes).

## Core Architecture Built

**Three tiers**:
- Tier 1 "Fix the Bleeding" — $297 (top 3 urgent issues, 48-72 hours)
- Tier 2 "Fix Everything" — $697 (full report implementation, 5-7 days)
- Tier 3 "Enterprise Partnership" — $1,997/month (ongoing monthly AI team)

## Key Pricing Insights

- Below $300 = single-decision-maker territory for most small business owners (no approval needed)
- Below $1,000 = still individual decision-maker for most
- $1,997/month monthly retainer is the LTV engine — one Tier 3 = 80 Tier 1 on 12-month LTV ($23,964)

## Most Powerful Conversion Argument

The "broken Stripe links" frame. It is not abstract — it means every purchase attempt fails. Every day of inaction = more lost sales. This visceral, quantifiable urgency is the strongest possible upsell hook. Use it directly: "Your checkout is broken. Customers who tried to buy today got an error. We can fix this in 48 hours for $297."

## Revenue Model Per 100 Analysis Customers

Expected blended upsell conversion:
- Tier 1 at 12-18%: $3,564-$5,346
- Tier 2 at 6-10%: $4,182-$6,970
- Tier 3 at 2-4%: $3,994-$7,988
- Total upsell: $11,740-$20,304
- Plus analysis: $4,700
- **Total per 100 customers: $16,440-$25,004**
- **Revenue per analysis customer: $164-250 average**
- This means CAC tolerance = $50-80 per customer acquisition

## Primary Competitive Differentiator

"Closed-loop AI" — the same agents that diagnosed the issues execute the fixes. Traditional agencies cannot offer this. No separate contract, no handoff, no implementation gap. Unique moat.

## Biggest Friction Point

Credential access. The solution is limited-permission contractor accounts — customers create a restricted account that can be deleted after work is complete. This eliminates the #1 objection without compromising the service.

## Upgrade Path Architecture

- Tier 1 → 30-day "what changed" email → Tier 2 pitch
- Tier 2 → 60-day GA4 performance report → Tier 3 pitch
- The upgrade path is where the compounding revenue lives
