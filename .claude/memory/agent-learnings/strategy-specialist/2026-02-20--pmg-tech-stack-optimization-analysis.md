## Key Findings

### Immediate Cuts (Tier 1 - no disruption)
1. **Zoom ($90.58/mo)** - Fully redundant with G-Suite Google Meet. Cancel immediately.
2. **Trello ($24/mo)** - Fully redundant with Bitrix24 project management. Cancel immediately.
3. **Apify OR Phantombuster** - Both do web scraping. Keep one:
   - Phantombuster ($69): LinkedIn automation focus, non-technical users
   - Apify ($39): Custom scraping, cheaper at scale

### Seat Audit Priority
- G-Suite at $268.80/mo suggests 15-22 seats for a 4-person team - likely stale accounts
- Admin audit at admin.google.com can reveal quick seat reductions ($50-100/mo savings)

### Protect These Tools
- Instantly: Revenue-generating outbound tool - don't cut based on cost alone
- Bitrix24: Core CRM + PM backbone
- Semrush: Justified if client SEO deliverables exist
- Figma: Core agency design tool

### AI Replacement Opportunities (Aether)
- Aether can partially replace: Apify custom scripts, make.ai automations, ChatGPT, Canva banners
- Not a full replacement yet - human workflow convenience matters
