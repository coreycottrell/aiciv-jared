**Date**: 2026-02-11

## Purpose

Shorthand commands for fast communication between Jared and Aether. Reduces typing, speeds up workflows.

## Core Commands

| Short | Meaning |
|-------|---------|
| D | Delegate to agent |
| R | Research only (don't build) |
| B | Build/implement |
| F | Fix this |
| P | Push to git |
| S | Status update |

D:web! Pop-Tarts analysis            → Urgent web research
