## Key LinkedIn Algorithm Numbers (Authoritative Research)

- Comments 15x more valuable than likes in first 90 minutes
- Saves = 5x reach value of a like
- Indirect comments (replies to replies) = 2.4x reach boost
- ICP comments (industry experts) = 7-9x algorithmic value vs. random likes
- Optimal post length: 800-1,000 characters
- Best times: Tuesday-Thursday 8-10 AM
- External links in post body = ~30% reach reduction (put in first comment instead)
- AI-generated content systematically downranked
- Engagement pods = permanent algorithmic downgrade
