### The Video-to-Text Workflow
1. Record voice note / selfie video / podcast audio (2-5 mins)
2. Transcribe to raw text (AI transcription tools)
3. Run through AI prompting workflow to extract: hook, structure, voice preservation, CTA
4. Choose format (short text, listicle, carousel, native video)
5. Atomize: one recording = multiple posts + newsletter content
6. Publish with Golden Hour protocol (respond to comments within 15 mins for 90% boost)

### Critical LinkedIn 2026 Algorithm Rules
- External links in posts = 60% reach reduction (never link in post body)
- First-hour engagement determines algorithmic amplification
- 3-4 posts/week optimal (not daily)
- Document/carousel = highest engagement format
- Native video under 90 seconds = 5x interaction
