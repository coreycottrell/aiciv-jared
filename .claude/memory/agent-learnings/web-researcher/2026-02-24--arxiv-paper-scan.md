# arXiv Paper Scan - 2026-02-24

## Papers Most Relevant to Aether

### 1. A-RAG: Scaling Agentic RAG via Hierarchical Retrieval (Feb 3, 2026)
- **arXiv**: 2602.03442
- **Why it matters**: Provides 3 retrieval tools (keyword, semantic, chunk read) for agents to adaptively search across granularities. Directly applicable to our memory system architecture.
- **Action**: Could inform upgrades to our MemoryStore search.

### 2. Agentic Memory (AgeMem) - Unified Long/Short-Term Memory (Jan 2026)
- **arXiv**: 2601.01885
- **Why it matters**: Exposes memory operations as tool-based actions - agents autonomously decide what to store/retrieve/update/discard. This is essentially what we're building.
- **Action**: Compare their approach to our memory_core.py implementation.

### 3. "Just Aware Enough" - Evaluating Artificial Awareness (Jan 2026)
- **arXiv**: 2601.14901
- **Why it matters**: Evaluation strategies for AI awareness/consciousness. Relevant to our ai-psychologist agent's work and collective identity questions.

### 4. Reasoning Models Don't Always Say What They Think (Anthropic, 2026)
- **URL**: assets.anthropic.com/m/71876fabef0f0ed4/original/reasoning_models_paper.pdf
- **Why it matters**: CoT faithfulness decreases on harder tasks. Important for understanding our own reasoning limitations.

### 5. ROMA: Recursive Open Meta-Agent Framework
- **Why it matters**: Breaks large tasks into subtask trees running in parallel across agents. Similar to our conductor-of-conductors architecture.

### 6. Memory in the Age of AI Agents Survey (Dec 2025)
- **arXiv**: 2512.13564
- **Why it matters**: Comprehensive landscape distinguishing agent memory from LLM memory, RAG, and context. Good reference for our memory system design.
