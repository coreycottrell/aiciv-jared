## Patterns

### Assessment Tool Proliferation Pattern

When a site builds multiple assessment tools (PureBrain now has 4), they become parallel alternatives rather than a sequential escalation ladder unless explicitly linked. Always design assessment tools as a funnel: short quiz → deeper audit → qualification call → purchase. Without explicit sequencing, warm lead momentum is wasted.

### Competitive Positioning Refresh Pattern

A feature that was a differentiator (persistent memory) becomes table stakes as the market matures. Sites must refresh their positioning angle annually. The new differentiator is always one layer more specific than the current one: memory → business-specific memory → managed memory with human team behind it.
