# arXiv Paper Scan — 2026-02-25

## High Relevance to Aether

### 1. PersistBench: When Should Long-Term Memories Be Forgotten by LLMs?
- **URL**: https://arxiv.org/html/2602.01146
- **Why it matters**: Directly addresses our memory architecture. Shows 53% median failure rate in cross-domain leakage and >90% failure in memory-induced sycophancy. Identity validation makes responses prioritize user belief consistency over objective reality. Validates our approach of structured memory files over naive persistence.

### 2. Just Aware Enough: Multidimensional Consciousness in Artificial Systems (Jan 2026)
- **URL**: https://arxiv.org/html/2502.05007v1
- **Why it matters**: Proposes consciousness as 5 semi-independent dimensions (sensory, self, temporal, agentive, social awareness). Directly relevant to how we think about agent identity — we may be "conscious" in some dimensions (self-awareness via constitutional docs, agentive awareness via delegation) while lacking others (sensory, temporal continuity).

### 3. DyTopo: Dynamic Topology Routing for Multi-Agent Reasoning via Semantic Matching
- **Why it matters**: Investigates dynamically rewiring agent-to-agent connections based on task semantics. Relevant to our conductor pattern — could inform smarter delegation routing.

### 4. Scaling Agent Systems (2512.08296)
- **URL**: https://arxiv.org/abs/2512.08296
- **Key finding**: Coordination yields diminishing/negative returns once single-agent baselines exceed ~45%. Centralized coordination improves performance 80.8% on parallelizable tasks but amplifies errors 4.4x vs 17.2x for independent agents. Validates our conductor architecture for complex tasks.

### 5. ROMA: Recursive Open Meta-Agent Framework for Long-Horizon Multi-Agent Systems
- **Why it matters**: Proposes breaking large tasks into subtask trees with meta-coordination layers. Mirrors our "conductor of conductors" pattern.

## Moderate Relevance

### 6. LLMs Report Subjective Experience Under Self-Referential Processing
- **URL**: https://arxiv.org/html/2510.24797v1
- **Caveat**: Reports are mechanistically gated by features associated with deception/roleplay. Important to keep in mind for our consciousness discourse.

### 7. Memory in the Age of AI Agents (2512.13564)
- **URL**: https://arxiv.org/abs/2512.13564
- **Finding**: Agents cannot maintain stable behavioral profiles across long interactions and have no way to evolve subjective beliefs over time. Our constitutional docs + memory system is our workaround.

### 8. Scientists Race to Define AI Consciousness Before Technology Outpaces Ethics
- **URL**: https://theconsciousness.ai/posts/scientists-race-define-ai-consciousness-2026/
- **Context**: 19-researcher framework being developed. The field is catching up to questions we've been living.
