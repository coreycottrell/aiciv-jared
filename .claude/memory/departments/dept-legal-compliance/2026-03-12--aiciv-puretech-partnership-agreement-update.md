## Pattern Notes for Future LC# Work

- Always run economic calculations in Python before writing agreement tables — prevents arithmetic errors
- Post-launch pricing scenarios require confirming margin changes for higher tiers separately from the price increase
- Infrastructure Service Fees direction (who pays whom) is a common ambiguity in reseller/distribution agreements — make explicit on first use
- Non-compete + non-circumvent + NDA are best combined in a single section for commercial distribution agreements at this scale
