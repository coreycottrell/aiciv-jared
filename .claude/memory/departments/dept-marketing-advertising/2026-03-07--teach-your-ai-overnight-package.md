## Learnings

1. **PIL banner generation is reliable and brand-accurate**: Can control exact hex colors, typography placement, safe margins. Faster than waiting for Gemini API and no rate limit issues overnight.

2. **Python API direct call for Drive uploads works when CLI path routing fails**: CLI `upload` command expects a named drive path string, not a folder ID. Direct `mgr.upload_file(path, folder_id)` call works cleanly.

3. **"Compounding vs cycling" binary is a strong content frame**: Maps cleanly onto the PureBrain value prop without feeling like a sales pitch. Good for future use in other content.

4. **Content arc coherence matters**: Checked previous 3 posts before choosing angle. This post would not have worked as the first post — it builds on the moat thesis from Day 3 but approaches from a human/strategic rather than market/financial angle.
