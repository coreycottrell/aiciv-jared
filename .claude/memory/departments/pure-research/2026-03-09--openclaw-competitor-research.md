## Key Facts About OpenClaw

- Open-source MIT-licensed self-hosted AI gateway framework
- Creator: Peter Steinberger (PSPDFKit founder)
- Previously called: Clawdbot -> Moltbot -> OpenClaw
- Hosted at: openclaw.ai
- Hostinger partnership: co-marketing page offering pre-configured VPS with OpenClaw

## Critical Strategic Event (Feb 14, 2026)
- Steinberger joined OpenAI to work on "bringing agents to everyone"
- OpenClaw transitioning to foundation structure (open source, community-governed, OpenAI-sponsored)
- Foundation transitions carry significant momentum risk for one-person projects
- PureBrain positioning opportunity: "stable company vs uncertain foundation"

## Product Architecture
- Self-hosted gateway connecting AI models to messaging apps
- Supports: Claude, GPT, Gemini, DeepSeek, Mistral, local Ollama models
- Messaging channels: WhatsApp, Telegram, Discord, iMessage, Signal, Slack + 7 more
- 50+ integrations including browser control, file system, terminal, calendar, smart home
- Skills marketplace: ClawHub (community-built JavaScript skills)

## Pricing Reality
- Software: FREE (MIT license)
- VPS hosting (Hostinger): ~$9-15/month
- AI API costs: ~$5-30/month (variable)
- Total: $14-45/month + setup time + ongoing maintenance
- No support, community-only

## Target Audience
- Developers and technical power users
- Privacy-conscious individuals wanting local data control
- NOT designed for non-technical users (creator's stated gap)

## PureBrain Competitive Advantages vs OpenClaw
1. Zero setup (vs 2-8 hours + ongoing maintenance)
2. Deep persistent memory (vs session-level only)
3. Professional/business workflow focus (vs personal automation)
4. Predictable pricing (vs variable API costs)
5. Customer support (vs community forum only)
6. Stable company (vs uncertain foundation transition)
7. Non-technical accessible (vs developer-only)

## What to Learn From OpenClaw
1. Messaging-native access (WhatsApp/Telegram) — evaluate for PureBrain Telegram bot
2. "Does things" framing is stronger than "AI assistant" — apply to copy
3. ClawHub skills marketplace model for network effects
4. Privacy/local data messaging resonates with technical audience
