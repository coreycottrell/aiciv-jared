# purebrain.ai Full Site Audit
**Date**: 2026-03-09
**Type**: analysis
**Topic**: purebrain.ai comprehensive site audit — SEO, UX, conversion, performance

## Key Findings

### Critical Gaps (fix immediately)
1. **Zero social proof** — no testimonials, logos, case studies on ANY page. Fatal for $149+/mo product.
2. **No pricing page** — /pricing/ returns 404. Visitors who want pricing bounce.
3. **GIF background** — Pure-Brain-Vid-3.gif used as full-page video background. Should be mp4/webm.

### Site Inventory
- 83+ published pages, 20 blog posts
- 13 comparison pages (strong SEO surface)
- 8+ internal/test pages publicly accessible (security + SEO risk)
- Blog categories: AI Strategy (7), AI Insights (6), For Individuals (3), For Teams (3), Agentic AI (2)

### SEO Issues
- FAQPage structured schema missing (accordions exist but not schema-tagged)
- Canonical tags not confirmed on key pages
- Duplicate tags ("AI strategy" vs "AI Strategy")
- "enterprise AI" and "AI adoption" tags have 0 posts despite high value
- CSS bloat: 6,000+ lines inline on multiple pages

### Conversion Issues
- Pricing not visible on homepage (meta says $149/mo but not on page)
- Invitation page has no visible CTA or form
- Why PureBrain page has zero testimonials
