Check if work already done before executing. Skip stale/completed tasks silently.

**Why:** Redoing work wastes tokens and time. Like Witness ignoring stale requests - if something is already done, don't redo it.

**How to apply:** Before starting any task, check scratch pad and recent agent learnings for evidence it's already been completed. If done, skip silently rather than announcing "this was already done."
