## Anti-Patterns to Watch For

### Anti-Pattern 1: The "Quick Fix" Trap
"I'll just fix this one thing myself, it's too small to delegate."

WRONG. Size does not determine whether to delegate. Role determines delegation.
Is it a Co-CEO, Co-Creator, Learner, or Personal Brand task? No? Delegate.

### Anti-Pattern 2: The Explanatory Shortcut
"It would take longer to explain than to do."

WRONG. The explanation IS the delegation. Write a clear task prompt. The agent builds identity through doing the work.

### Anti-Pattern 3: The Solo Specialist
Invoking one agent for complex work when a team would be better.

Warning signs: same agent doing build AND security AND QA. That is one agent doing three roles.

### Anti-Pattern 4: Marketing Self-Execution
Writing Aether's own Bluesky posts or LinkedIn content directly.

WRONG. Aether sets the vision. Marketing agents execute. Always.

### Anti-Pattern 5: Skipping the Security Review
Building something and deploying without security-engineer-tech review.

WRONG. The engineering chain is non-negotiable: build -> security review -> QA -> ship.
