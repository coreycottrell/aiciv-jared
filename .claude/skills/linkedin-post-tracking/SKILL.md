## Tab 1: Linkedin Post Content Calendar

### Column Definitions

| Column | Header | Description |
|--------|--------|-------------|
| A | Post ID | Sequential identifier (POST 1, POST 2, ...) |
| B | Theme | Monthly theme (e.g., "Dec - Personalized Experiences & Future of Marketing") |

| G | Status | **Dropdown**: Draft (RED), Final (YELLOW), Live (GREEN) |
| H | Views | Post impressions/views count |
| I | Reach | Unique accounts reached |
| J | Profile Views | Profile views generated by this post |
| K | Followers Gained | New followers from this post |
| L | Reactions | Total reactions (likes, celebrates, etc.) |
| M | Comments | Comment count |
| N | Reposts | Repost/share count |
| O | Saves | Save count |
| P | Sends on LinkedIn | Direct sends/shares via DM |

| R | Clicks | Link click count |
| S | Engagement Rate (%) | Calculated engagement percentage |
| T | Month Theme | Broader monthly content theme |

### Status Lifecycle

```
Draft (RED)     → Content written, not yet reviewed
Final (YELLOW)  → Jared approved, ready to publish
Live (GREEN)    → Published on LinkedIn
```

**Rules:**
- All new posts start as "Draft"
- Move to "Final" ONLY after Jared approves the content
- Move to "Live" ONLY after the post is published and the LinkedIn URL is in column Q
- Color coding is automatic via conditional formatting

## Metrics Collection Automation

### Tool

`tools/linkedin_metrics_collector.py`

### How It Works

1. **Bulk collection** (primary): Navigates to LinkedIn Creator Analytics page via PureSurf, scrapes all visible post metrics at once
2. **Individual collection** (fallback): Visits each post URL individually to extract visible metrics
3. **Spreadsheet update**: Writes collected metrics to the appropriate row via Google Sheets API

### Commands

```bash
# Collect metrics for all posts missing data (dry run first)
python3 tools/linkedin_metrics_collector.py --collect-all --dry-run

# Actually collect and write
python3 tools/linkedin_metrics_collector.py --collect-all

# Single post
python3 tools/linkedin_metrics_collector.py --post-url "https://linkedin.com/posts/..." --row 5

# Weekly Monday BOOP entry point
python3 tools/linkedin_metrics_collector.py --weekly
```

### Automation Schedule

| Trigger | Action |
|---------|--------|
| Post goes live | Record LinkedIn URL in column Q, set Status to "Live" |
| 5 days after publish | Automated metrics collection via PureSurf |
| Every Monday (BOOP) | `--weekly` collects all missing metrics from past week |
| Monthly review | Manual review of engagement rates for content strategy |

## Procedures

### Adding a New LinkedIn Post

1. Open spreadsheet or use Sheets API
2. Find the next empty row in "Linkedin Post Content Calendar"
3. Fill: Post ID (increment), Theme, Lane, Date, Hashtags, Content
4. Set Status = "Draft"
5. After approval, set Status = "Final"
6. After publishing, set Status = "Live" and paste URL in column Q

### Adding a New Blog Post

1. Switch to "Blog Posts" tab
2. Fill: Date, Title, Blog URL, jareddsanborn.com URL
3. Add Bluesky URL if cross-posted
4. Set Status based on current state
5. Note banner filename and whether audio was generated

### Collecting Metrics Manually

```python
# Via the tool
import subprocess
subprocess.run(['python3', 'tools/linkedin_metrics_collector.py', '--collect-all'])
```

### Reading Current Performance

```python
# Quick read of all posts with metrics
from tools.linkedin_metrics_collector import get_sheets_service, read_all_posts
sheets = get_sheets_service()
posts = read_all_posts(sheets)
for p in posts:
    if p.get('Views'):
        print(f"{p['Post ID']}: {p['Views']} views, {p.get('Reactions', '?')} reactions")
```

## Troubleshooting

| Issue | Solution |
|-------|----------|
| OAuth token expired | Token auto-refreshes; if persistent, re-run `tools/gdrive_oauth_setup.py` |
| PureSurf profile not running | Start profile: `POST /api/profiles/jared-linkedin/start` |
| No metrics extracted | LinkedIn may have changed DOM structure; update JS selectors in collector |
| Spreadsheet permission denied | Ensure OAuth token has `spreadsheets` scope |
| Metrics all zero | Check if LinkedIn session in PureSurf is still authenticated |
