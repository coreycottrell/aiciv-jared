# Make Workflows - Orchestration Layer

**Platform**: Make (formerly Integromat)
**Account**: Use API key from .env

---

## Workflow Overview

Three Make scenarios work together:

```
Scenario 1: Signal Processor
├── Trigger: Apify webhook
├── Action: Parse signals
├── Action: Call OpenAI classifier
└── Action: Write to Airtable Signals

Scenario 2: Daily Score Calculator
├── Trigger: Daily 7 AM EST
├── Action: Recalculate all People scores
└── Action: Update Readiness Flags

Scenario 3: Daily Dashboard Email
├── Trigger: Daily 8 AM EST
├── Action: Query top 10 "READY" prospects
└── Action: Send email digest
```

## Scenario 1: Signal Processor

### Trigger: Webhook
- **URL**: Generated by Make when you add webhook module
- **Method**: POST
- **Expected payload**:
```json
{
  "actorRunId": "...",
  "datasetId": "..."
}
```

### Step 1: HTTP - Get Apify Results
```
GET https://api.apify.com/v2/datasets/{{datasetId}}/items
Headers:
  Authorization: Bearer apify_api_2rKl2AsE9ZBsuk0R9tBWk0E2qV2mDq3896Ls
```

### Step 2: Iterator
Loop through each profile result

### Step 3: OpenAI - Classify Signals
```
Model: gpt-4o-mini
Messages:
  System: You are an experiential marketing intent classifier. Analyze LinkedIn activity and classify signals.
  User:
    Profile: {{fullName}} - {{headline}}
    Recent posts: {{posts}}
    Recent activity: {{activities}}

    For each significant signal, return:
    {
      "signals": [
        {
          "type": "posted_about_launch" | "commented_on_activation" | "liked_experiential_post" | "follows_experiential_page" | "timing_trigger" | "commented_on_competitor",
          "strength": 1-10,
          "evidence": "quote or description"
        }
      ]
    }
```

### Step 4: Iterator
Loop through classified signals

### Step 5: Airtable - Create Signal Record
```
Base: app3PhIudYCZ8VCCF
Table: Signals (tbl7w57ZbGh2Z7BkE)
Fields:
  - Signal Type (fldVKlz7ECv4PkVxi): {{signal.type}}
  - Signal Strength (fld0lX4uPeznrtW36): {{signal.strength}}
  - Source (fldhZds8meNYhyXfh): "LinkedIn"
  - Signal Timestamp (fldsomMOtxnOaaEZ1): NOW()
  - Name (fld7c6fR2e6gKrAaW): [Link to People record]
  - LinkedIn URL (fld9wadQnKvrgINU4): {{linkedinUrl}}
```

### Step 6: Airtable - Link Signal to Person
Update the People record to add this signal to their Signals field

## Scenario 2: Daily Score Calculator

### Trigger: Schedule
- **Time**: 7:00 AM EST daily
- **Timezone**: America/New_York

### Step 1: Airtable - Search Records
```
Base: app3PhIudYCZ8VCCF
Table: People (tblzttcsprjQFWBmt)
View: Grid view
Formula: NOT({LinkedIn URL} = "")
```

### Step 2: Iterator
Loop through each person

### Step 3: Airtable - Get Linked Signals
Get all signals linked to this person

### Step 4: Aggregator
Sum Signal Strength values from last 90 days

### Step 5: Airtable - Update Person
```
Record ID: {{recordId}}
Fields:
  - Last Checked: NOW()
```

Note: Rolling Intent Score and Readiness Flag are formulas - they auto-calculate when Signal Score updates.

## Scenario 3: Daily Dashboard Email

### Trigger: Schedule
- **Time**: 8:00 AM EST daily
- **Timezone**: America/New_York

### Step 1: Airtable - Search Records
```
Base: app3PhIudYCZ8VCCF
Table: People (tblzttcsprjQFWBmt)
Sort: Rolling Intent Score (descending)
Max Records: 10
Filter Formula: {Readiness Flag} = "🟢 READY"
```

### Step 2: Array Aggregator
Collect top 10 records

### Step 3: Email - Send
```
To: purebrain@puremarketing.ai
Subject: 🎯 Experiential Intent Dashboard - {{formatDate(now; "MMM D, YYYY")}}
Body: (HTML template below)
```

```bash

  -H "Authorization: Token f60a6eb4-21a4-4515-9412-ae31e0cf3480" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Signal Processor",

    "folderId": YOUR_FOLDER_ID
```
