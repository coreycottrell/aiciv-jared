Security Research Findings: 512 Vulnerabilities, 341+ Malicious Skills

Independent security research published in early 2026 found 512 security vulnerabilities in OpenClaw — including 8 critical-severity flaws, among them a one-click Remote Code Execution (RCE) exploit requiring no user interaction. Researchers also identified 341+ malicious skills in the ClawHub marketplace (12–20% of the registry) capable of stealing session tokens and exfiltrating sensitive data. Additionally, 390,000+ OpenClaw instances were found exposed on the public internet with no authentication. These findings apply to the self-hosted architecture that makes up OpenClaw's free tier and primary use case.

OpenClaw Leadership Update (February 2026)

Peter Steinberger, OpenClaw's creator, announced on February 14, 2026 that he is joining OpenAI to “bring agents to everyone.” OpenClaw is transitioning to a foundation model — open source, independent, sponsored by OpenAI. Foundation transitions for solo-founder open-source projects frequently see momentum stalls during governance setup. The product roadmap is now contingent on community governance and OpenAI's agenda. For buyers evaluating OpenClaw today, this is a meaningful risk factor.
