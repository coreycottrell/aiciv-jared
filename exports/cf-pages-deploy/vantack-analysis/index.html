Section 01

What Is Vantack?

Vantack is a cryptocurrency payment orchestration platform that makes stablecoin payments accessible and practical for high-volume businesses. Think of it as the Stripe for the crypto-native world — but with on/off ramps, global reach, and a non-custodial architecture that protects both merchant and buyer.

50K+ Payments Processed
100+ Businesses Using
178+ Countries Covered
40+ Real-Time Payment Markets
4.99 Average Rating / 5.0
⚡
Instant Settlement
Wallet-to-wallet transfers settle in minutes, not days. No chargebacks, no holds. Merchants get paid and move on.
🔓
Non-Custodial Architecture
Merchants can only charge what customers have pre-approved. Trustless, transparent, fraud-resistant by design.
🔄
On/Off Ramps Built In
Automatic fiat conversion through MoonPay and Transak. Accept crypto, receive fiat — or hold stablecoins. Merchant's choice.
📈
Recurring Subscriptions
Subscription billing in stablecoins (USDC/USDT). Auto-pull from approved wallets. Customers can manage or cancel anytime.
🌐
Global by Default
40 currencies, 178+ countries, EVM-compatible now with Solana coming. Africa is a priority expansion market — major underserved opportunity.
🛡
Enterprise-Grade Security
SOC 2 Type 1 & 2 (2024), ISO 27001/27017/27018, WebAuthn, DASP License. More certifications in progress.

Positioning: Vantack is not trying to replace PayPal for everyone. It's building the payment infrastructure for the businesses PayPal cannot or will not serve well — global, crypto-native, high-volume, emerging market operators who need speed, transparency, and lower fees without the bureaucracy of traditional fintech.

Vantack vs PayPal: The Real Comparison

PureBrain currently uses PayPal for all paid tiers: Insiders ($74.50), Awakened ($197), Partnered ($579), Unified ($1,089). Here is what that actually costs us — and what switching would look like.

Dimension PayPal (Current) Vantack Edge

2.29% + $0.09 (card reader)		 Stablecoin rates — no disclosed %; likely 0.5–1.5% for crypto (market standard) or custom negotiated rate as partner Vantack
International Fees +1.5% cross-border + currency conversion Global stablecoin is inherently borderless — no cross-border surcharge Vantack
Chargeback Risk Full chargeback exposure; PayPal disputes often favour buyers Crypto is final — no chargebacks. Pre-approved amounts only. Zero dispute exposure. Vantack
Settlement Time 1–3 business days to bank Minutes, wallet-to-wallet Vantack

Buyer Friction Logged-in flow. Most people already have PayPal. Requires crypto wallet. Higher barrier for non-crypto users. PayPal
Subscription Billing Supported (but complex) Native stablecoin recurring billing built-in Vantack
API Quality REST API — mature but dated. Complex webhook infrastructure. Modern API. Designed for developer integration alongside existing stacks (Stripe/PayPal/Square compatible layer) Tie
Partner Relationship No partner relationship. Pure vendor. Active partner — co-marketing, co-growth, strategic alignment Vantack
WhiteLabel Option None Whitelabel-ready for PSPs and large operators Vantack

Reasons to Switch to Vantack

  • Lower fees — especially at our price points ($197–$1,089), fee savings per transaction are material
  • No chargebacks — removes our biggest fraud and dispute risk
  • No PayPal account freeze risk — they have frozen many SaaS businesses overnight
  • Partner-aligned pricing — as a named partner, we can negotiate custom rates
  • Instant settlement — cash flow improvement, especially for enterprise tier
  • AiCIV partnership agreement already commits to this migration
  • Positions PureBrain as a crypto-forward AI company — aligns with 2026 market narrative
  • Africa expansion opens new customer geography for PureBrain

Risks & Complications

  • Buyer familiarity gap — most PureBrain customers do not have crypto wallets
  • On-ramp friction — buying USDC/USDT to pay is a multi-step process for crypto newcomers
  • Pricing must be displayed in stablecoin amounts or require USD-to-crypto conversion UI
  • Integration effort — payment pages need to be rebuilt or extended
  • Vantack is early-stage (100+ businesses) — platform maturity risk
  • No disclosed pricing publicly — need direct conversation with Max Rosen to confirm rates
  • Regulatory grey zone in some states/countries for stablecoin payments

Recommended approach: Do not fully replace PayPal immediately. Instead, run Vantack as a parallel payment option — offer both methods at checkout. This lets us test conversion rates, build buyer familiarity, and capture crypto-native customers (who likely have higher LTV) while keeping the PayPal safety net. Full migration when Vantack handles 30%+ of volume organically.

Section 05

Integration Plan

The technical and commercial path to running Vantack alongside (and eventually replacing) PayPal on PureBrain's checkout pages. This is a phased migration, not a hard cutover.

Current State: PureBrain payment pages at /insiders/, /awakened/, /partnered/, /unified/ all run PayPal JavaScript SDK. Each page has a PayPal button that opens the PayPal checkout modal. We deploy to Cloudflare Pages (purebrain-staging). No server-side payment logic — it's all client-side PayPal SDK with server webhook confirmation via PureBrain Worker.

Phase 1 — Parallel Integration (Weeks 1–3)

Step 1: API Keys + Account Setup
Create Vantack merchant account, generate API keys, configure merchant wallet (USDC/USDT), set up fiat off-ramp preference via MoonPay/Transak integration in Vantack dashboard.
Step 2: Checkout Component
Build Vantack checkout button component for CF Pages. Vantack supports desktop and mobile browsers. Add alongside existing PayPal button with clear "Pay with Crypto (USDC/USDT) — Lower Fees" label.
Step 3: Webhook Confirmation
Integrate Vantack payment webhooks into PureBrain Worker (same pattern as PayPal). On payment confirmation, trigger the same onboarding flow — container creation, magic link, welcome email.

Phase 2 — AiCIV Settlement Automation (Week 2–4)

Daily Revenue Split
Automate the 60/40 (later 50/50) revenue split from PureBrain to AiCIV using Vantack's wallet-to-wallet transfer API. Replace manual PayPal sends with automated daily Vantack disbursements.
Segregated Account
Per partnership agreement Section 5.4, maintain a segregated wallet for AiCIV's share until automated payout is live. Vantack's non-custodial architecture makes this natural — both parties hold their own wallet.

Technical Dependencies

Vantack API Documentation
Need to review full API docs (not publicly accessible) before integration scoping. Request from Max Rosen on first call.
Customer Wallet Requirement
Customers need a crypto wallet (MetaMask, Coinbase Wallet, etc.) and USDC/USDT. Consider adding a "How to buy USDC" guide to the checkout page to reduce friction.
On/Off Ramp UX
Vantack integrates MoonPay and Transak for fiat conversion. Test the off-ramp flow so we know exactly how proceeds hit our bank account vs stablecoin wallet.

Technical flag for ST# team: Integration requires no WordPress — we are fully on CF Pages. The webhook pattern for Vantack should mirror the existing PayPal webhook → PureBrain Worker → container creation flow. Assign to Witness/Corey for backend Worker integration.
