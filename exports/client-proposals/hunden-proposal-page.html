Needs Analysis

Your org chart and agent summary document confirms this is not a tool-shopping exercise. Hunden knows what it wants to build. Five operational domains with distinct AI needs — each mapped to a targeted civilization.

Domain 01
Consulting Delivery
Analysts, PMs, EVPs. The analyst tier is the highest-ROI pilot target. Research-to-insight-to-exhibit workflows where AI civilizations compound fastest. PMs and Directors need cohesion and QC agents trained on Hunden's deliverable structure, brand standards, and approved narrative blocks.
Domain 02
BizDev / Sales / Marketing
Proposals, compliance matrices, content flywheel, RFP lead capture. One of the highest dollar-per-hour leverage ratios in consulting. A proposal that takes 12 analyst hours becomes a 2-hour QA exercise when an AI civilization drafts the first pass from precedent models.
Domain 03
Operations / Admin / Finance
Invoicing orchestration, collections, KPI dashboards, inbox cleanup. Lower-creativity, higher-volume workflows. An AI civilization here creates margin by removing administrative overhead from senior staff — freeing them for advisory work.
Domain 04
Product / Tech (Horizons + ProCloud)
Support triage, onboarding, QA/regression, release notes, product ops. Needs an AI civilization that understands the product deeply — modes, guardrails, user types — supporting both internal and external users with appropriate access controls.
Domain 05
Expansion Areas
HR, Billing, Marketing, Sales, State of the Industry Research Database, Custom Mapping. Greenfield opportunities identified for CCW consideration. PureBrain handles all within a unified ecosystem rather than requiring separate point solutions.
Cross-Cutting
Client-Facing Layer
External clients using Hunden Horizons need board-ready intelligence with enforced guardrails — scenario analysis, collaboration workflows, and Q&A with Client Mode restrictions. This is a product offering, not an internal tool.

The Hunden OS Layer

Every agent Hunden has designed inherits from a shared canon: brand standards, deliverable structures, approved narrative blocks, precedent models, assumption ranges, and QC rubrics. This is precisely what PureBrain's shared memory architecture provides at the civilization level. The Hunden OS is not a document — it is the persistent memory substrate that all agents draw from. PureBrain builds this as infrastructure, not as a prompt.

Hunden OS Core — Shared Infrastructure Layer

Not a seventh civilization — the connective tissue that makes all six civilizations coherent. Hunden OS as living infrastructure, not a static document.

Shared Memory Store
Brand standards, approved narrative blocks, deliverable templates, precedent model ranges, assumption ranges, QC rubrics — persistent and accessible by every agent across all civilizations.
Permissions Architecture
Client Mode vs Studio Mode enforced at the infrastructure level, not just per-agent. Client Mode agents cannot access Studio power tools regardless of user request.
Cross-Civilization Routing
When BizDev needs a market comp that HCI already researched, the request routes without duplication — no duplicate work, no manual handoffs.
Universal Quality Gate
QC rubric runs on all outputs before delivery — whether an analyst exhibit, a proposal draft, or a client-facing Horizons report. Quality is infrastructure, not afterthought.
