## COPY NOTES FOR JARED

### Tone Rationale
This copy reads like an invitation to a private club, not a sales page. The exclusivity is real (25 spots, password-protected, invite-only), so the copy leans into that rather than manufacturing artificial urgency. The tone is confident and direct — the kind of page that earns trust because it doesn't oversell.

### Michael Hancock Usage
I used Michael by name and role. If he hasn't given explicit permission to be quoted by name on a public-facing (even if password-protected) page, recommend either:
- Getting his explicit go-ahead via a quick message
- Replacing with: "Michael H., General Counsel at a multi-firm practice" with exact quote intact

The quote itself is paraphrased from the context you provided. If he has an actual verbatim quote, swap it in — it will convert better.

### The Bonded Tier Emphasis
The copy consistently channels attention toward Bonded ($149/mo) as the recommended tier without dismissing the others. The "RECOMMENDED FOR BUSINESS" badge, the strikethrough pricing, and the "SAVE $47/MO" callout all work together to make Bonded feel like the obvious choice for the target audience (executives, founders, professionals).

### The Awakening Section
The "Deep Brief" and "named AI partner" language is drawn from PureBrain's core differentiation — permanent memory, compounding value, named relationship. This is the moment that sets PureBrain apart from every AI tool the prospect has tried. The copy makes it experiential rather than technical.

### CTA Link
All primary CTA buttons should route to the pay-test-2 page with Bonded tier pre-selected where possible. If the page supports direct-to-tier linking, Bonded = $149/mo.
