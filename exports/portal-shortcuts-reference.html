Everything you can do from the command line, keyboard, and voice overlay

/ Slash Commands (Quick Fire)

Type these in chat or click them in the Quick Fire sidebar. Fully customizable — add your own in Settings.

CommandLabelWhat It DoesType
/boop BOOP Trigger scheduled automation cycle Default
/sprint-mode Grounding Enter grounding/sprint mode for focused work Default
/status Status Check system status and active processes Default
/compact Compact Show compact status view Default
/intel-scan Intel Run AI industry intelligence scan Default
/rubber-duck Duck Start rubber duck debugging session Default
/your-command Custom Add your own via Settings → New Command Custom
Adding Custom Commands

Settings (gear icon) → scroll to Quick Fire Commands → enter a label and the command text → press Enter or click "Add Command". Reorder with the up/down arrows, delete with the X.

Voice Overlay (HMI)

Hands-free voice interaction with your AI. Click the mic icon to open.

FeatureWhat It DoesHow To Use
Mic Button Toggle listening on/off Click the mic icon in the overlay
Conversation Mode Auto-listen after AI responds Opens automatically when overlay is active + mic on
Send Word Say a magic word to auto-send your message Set in overlay settings (e.g. "go eat that")
Engine: Browser Use built-in browser TTS Click BROWSER button
Engine: ElevenLabs Use ElevenLabs premium voices Click ELEVENLABS+ button (requires API key)
Gear Icon Open ElevenLabs settings Enter API key, select voice, test voice
AI Voice Dropdown Select which voice your AI speaks with Shows browser voices or ElevenLabs voices based on engine
Status Indicators STANDBY / LISTENING / PROCESSING / TRANSMIT Shows current voice system state

BOOP Automation

Schedule any slash command to run automatically on a timer.

SettingWhat It DoesWhere
Active Command Which slash command runs on each BOOP Settings → BOOP Config
Cadence Minutes between each BOOP (default: 30) Settings → BOOP Config

Chat Features

Built-in capabilities when chatting.

FeatureHow It Works
File Upload Drag & drop files into chat, or click the attachment icon. Files are sent with your message.
Image Upload Drag & drop images or paste from clipboard. AI can see and analyze images.
Reply to Message Click a message to quote-reply. Your reply includes context of what you're responding to.
File Downloads AI-generated files appear as downloadable cards with preview. Click to download.
Markdown Rendering AI responses render with full markdown: headers, bold, code blocks, lists, links.
Clickable Links All URLs in messages are automatically clickable (opens in new tab).
Search Ctrl+F to search through all chat messages. Enter for next match, Shift+Enter for previous.
