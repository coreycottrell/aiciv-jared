## REQUIRED ASSETS — Cannot Launch Without These

### 1. Product Logo / Thumbnail

**Purpose**: The first image anyone sees on Product Hunt. Appears in feed, notifications, and the listing.

| Spec | Value |
|------|-------|
| Dimensions | 240 x 240 pixels |
| Format | PNG |
| Background | Transparent (preferred) or white |
| File size | Under 1MB |
| Deadline | April 19, 2026 |

**Design requirements**:
- Legible at small sizes (it will appear as small as 40x40px in some views)
- Do NOT include the word "PureBrain" as text in the logo — the product name appears separately
- Use the PureBrain brand mark/orb icon, not the full logotype
- Test it at 40px, 80px, 120px, and 240px to verify legibility at all sizes
- Match the visual style already established on purebrain.ai

**Aether action**: Generate or export the brand icon in 240x240 PNG format. Cross-reference with existing logo assets in WordPress media library.

### 2. Gallery Screenshots (x5)

**Purpose**: The visual story of the product. These are what users browse to understand what PureBrain actually does before clicking through to the website.

| Spec | Value |
|------|-------|
| Dimensions | 1270 x 760 pixels (16:9 ratio) |
| Format | PNG or high-quality JPG |
| Count | 5 images minimum (Product Hunt shows up to 5) |
| File size | Under 5MB each |
| Deadline | April 26, 2026 |

**Required screenshots (in this order)**:

**Screenshot 1 — Assessment Start / Homepage**
- What to show: The assessment landing page or the opening question of the assessment
- Overlay text (top or bottom): "AI Adoption Assessment — Free, 10 minutes, scored"
- Why: This is the free entry point. Lead with what doesn't cost anything.

**Screenshot 2 — Assessment Results Page**
- What to show: A sample scored results page (use a real result if available, or a representative demo)
- Overlay text: "Specific scores. Specific gaps. A concrete roadmap."
- Why: Shows the output — what you get for completing the assessment. This is the "aha" moment visual.

**Screenshot 3 — Tool Stack Calculator**
- What to show: The calculator interface with tools entered and results showing
- Overlay text: "AI Tool Stack Calculator — Find what's redundant and what's missing"
- Why: Demonstrates a second distinct value proposition. Breadth of platform.

**Screenshot 4 — Migration Portal**
- What to show: The migration portal wizard or results view
- Overlay text: "Structured AI migration — not just 'switch tools'"
- Why: Third value proposition, shows this is a full platform not a single tool.

**Screenshot 5 — Overall Platform / Dashboard (or AI Partnership Audit)**
- What to show: Either an overview of all four tools, or the AI Partnership Audit interface
- Overlay text: "Complete AI strategy platform — assessment, tools, migration, partnership"
- Why: Reinforces the full scope. Leaves viewers with a "there's a lot here" impression.

**Design guidelines for all screenshots**:
- Consistent header/footer treatment (same font, same color overlay style)
- Dark overlay at bottom or top with white text (high contrast, legible)
- Do NOT include actual client data — use sample or anonymized data
- Browser chrome (URL bar, tabs) should be cropped out unless it adds context
- Test for legibility on a phone screen (many PH users are mobile)

### 3. Demo Video

**Purpose**: The single most important asset for a Product Hunt listing. Products with demo videos consistently outperform those without. This is where the "aha moment" lives.

| Spec | Value |
|------|-------|
| Format | MP4 |
| Length | 2:00 - 3:00 minutes (absolute maximum) |
| Dimensions | 1920 x 1080px (HD) |
| Captions | REQUIRED — many users watch muted |
| Audio | Clean, no background noise. Jared on mic. |
| Deadline | April 19, 2026 |

**Required elements**:
- Jared on camera (30-60 seconds) — makes it human, builds trust
- Product screen recording (90-120 seconds) — shows the actual tool
- Captions synced throughout
- End card with purebrain.ai URL and "Get Started Free" CTA

**See `launch-day-plan.md` for the full demo video script.**

**Production options**:
- DIY: Loom (free, easy, lower production quality)
- Mid: ScreenFlow or Camtasia (better quality, ~$200 software)
- Pro: Video editor ($500-1,500 for 3 min polished video)

**Recommendation**: Do it in Loom or similar first. If quality is acceptable, ship it. PH users care far more about authenticity and product quality than production value. A shaky but genuine walkthrough with Jared talking beats a slick corporate video every time.

## Pre-Upload Review Checklist

Before uploading any asset to Product Hunt:

**Logo**:
- [ ] Renders cleanly at 40px, 80px, 240px
- [ ] No white border artifact on dark backgrounds
- [ ] Brand-accurate colors

**Screenshots**:
- [ ] All are 1270x760px exact (verify in image properties)
- [ ] Text overlays are legible on mobile
- [ ] No actual client/user data visible
- [ ] Consistent visual style across all 5

**Demo Video**:
- [ ] Captions are accurate and synced
- [ ] No background noise
- [ ] Jared segment is well-lit and in focus
- [ ] Ends with clear URL + CTA
- [ ] Under 3 minutes

**Copy**:
- [ ] Tagline is under 60 characters
- [ ] Short description is under 260 characters
- [ ] No typos (proof once more day-of)
- [ ] Maker comment is personal, not corporate
