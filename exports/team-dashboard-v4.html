Optional: Connect a shared Supabase database so tasks sync across all team members in real-time.

Step 1: Go to supabase.com → New Project (free tier is fine).
Step 2: In your project, go to SQL Editor and run this:
create table tasks ( id text primary key, title text, description text, assignedTo text, delegation text, priority text, status text, deadline text, files text, dept text, cadence text, createdAt text, createdBy text );
Step 3: Go to Settings → API and copy your Project URL and anon key below.
Step 4: In Authentication → Policies, enable RLS and add a policy allowing all operations (for internal use).
