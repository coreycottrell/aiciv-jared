## Key Learnings Discovered Today

- **overflow-x: hidden kills position: sticky** - Use overflow-x: clip instead (confirmed in CSS spec)
- **Broad [class*="magic"] selectors** match too many elements and poison content visibility when using inherit values
- **WordPress plugin footer override must include ALL page IDs** - missing one causes white page even with correct content CSS
- **WordPress password form CSS must go in wp_head** - cannot be styled from inside wp:html content block
- **BOOP scheduler stagger is critical** - all tasks at same time = thundering herd problem
- **Nested HTML documents in WordPress = white page** - always extract style + body, never deploy full HTML
- **PayPal server-side verification is still missing** (HIGH security risk - page 826)
