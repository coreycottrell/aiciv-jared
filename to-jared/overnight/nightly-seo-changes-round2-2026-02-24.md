## Technical Notes

**Verification requires auth**: When checking `yoast_head_json.description` without the `-u Aether:APP_PASSWORD` header, some pages return cached empty responses due to Cloudflare caching. Always verify with authentication.

**apostrophe/special char pattern**: Descriptions with apostrophes must use `--data-raw` + Python `json.dumps()` to escape properly. Shell quoting with single apostrophes breaks JSON parsing. Safe pattern:

```bash
--data-raw "{\"post_id\": $pid, \"meta_key\": \"_yoast_wpseo_metadesc\", \"meta_value\": $(echo "$desc" | python3 -c "import sys,json; print(json.dumps(sys.stdin.read().strip()))")}"
```
